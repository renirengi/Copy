Task

Your task is to make a function that can take any non-negative integer as an argument and return it with its digits in descending order. Essentially, rearrange the digits to create the highest possible number.


         function descendingOrder(n){

         let myString=n.toString();

         let myArray=[];

         myArray = myString.split('');

         myArray.sort();

         myArray.reverse();

         myString=myArray.join('')

         return parseInt(myString);

         }

Custom video player

The project is a custom video player. The layout is based on HTML, styles are created using CSS, the slider is launched using JS (working with Dom-elements). The player can be controlled using hot keys: "Space" key - pause, "m" key (English) - disable / enable sound, key "f" - enable / disable full-screen mode. Added player control from the keyboard: "k" -pause or continue playback, "j"- rewind the video 10s backward and forward, respectively, "l" - rewind the video 10s forward, "0..9" - go to a certain point in the video, "." - jump to the next frame when playback is paused, "b" - jump to the next frame when playback is paused. The keyboard shortcuts work the same way as on YouTube. The keys work in two layouts: Russian and English.
